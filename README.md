# 🎮 Polly Challenge - 宝可梦挑战游戏

基于 math-challenge-grade4 创建的宝可梦主题挑战游戏。

## ✨ 特点

- 📚 **双题库系统**：数学挑战 + 科学挑战
- 🎯 **30 题闯关**：每答对 6 题获得技能卡奖励
- 🌟 **宝可梦奖励**：答对题目随机获得宝可梦（1025 只全收录）
- ✨ **闪光系统**：10% 概率出现闪光宝可梦，第 30 题必定闪光！
- 🎴 **技能卡系统**：
  - 求助爸爸：高亮正确答案
  - 排除错误：去除两个错误选项
  - 换题：更换当前题目

## 📊 数据完整性

**✅ 所有 1025 只宝可梦，100% 中文化！**

数据格式统一：**编号 + 名称 + 描述**

| 世代 | 编号范围 | 数量 | 描述格式 |
|------|---------|------|---------|
| 第一世代 | #1-151 | 151 只 | 详细介绍（保留原始数据）|
| 第二~九世代 | #152-1025 | 874 只 | 属性描述（格式：属性：XX/XX）|

**示例**：
- #1 妙蛙种子：「出生后背上就背着种子。种子会随着身体的成长而长大。」
- #392 烈焰猴：「属性：火/格斗」
- #1025 桃歹郎：「属性：毒/幽灵」

## 🎨 游戏玩法

1. 选择题库（数学挑战 / 科学挑战）
2. 回答 30 道题目
3. 答对获得宝可梦奖励
4. 每 6 题获得随机技能卡
5. 答错即失败，需重新开始
6. 第 30 题通关后获得特殊闪光宝可梦！

## 🔧 技术栈

- 纯前端实现（HTML + CSS + JavaScript）
- 宝可梦数据来源：
  - 中文数据：[42arch/pokemon-dataset-zh](https://github.com/42arch/pokemon-dataset-zh)
  - 图片：[PokeAPI Sprites](https://github.com/PokeAPI/sprites)

## 🚀 开始使用

直接打开 `index.html` 即可！无需服务器，无需构建。

## 📝 项目结构

```
polly-challenge/
├── index.html              # 主页面
├── game.js                 # 游戏逻辑
├── questions.js            # 数学题库
├── science-questions.js    # 科学题库
├── adventure-scenes.js     # 冒险场景文案
├── pokemon-data.js         # 宝可梦数据接口
└── pokemon-zh-data.js      # 宝可梦中文数据（1025 只）
```
